# stagbeetle

You see battle animations from Mother2 (a.k.a. Earthbound).

## Installation

Download from http://example.com/FIXME.

## Usage

It's pretty easy to run stagbeetle.

Download stag-x.xx.jar from [here](https://github.com/subaru45/stagbeetle),
and double-click it!
